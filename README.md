
# Focus Marketing Billing Software

A simple Streamlit billing software for generating editable Word invoices.

## Features

- Create invoices
- Add multiple services
- Auto GST calculation
- Download editable DOCX invoice
- Simple Streamlit UI

---

# Installation

## Step 1 - Install Python

Download Python:
https://www.python.org/downloads/

IMPORTANT:
Enable:
[x] Add Python to PATH

---

## Step 2 - Extract ZIP File

Extract the project folder.

---

## Step 3 - Open Terminal

Open terminal inside project folder.

---

## Step 4 - Install Requirements

Run:

pip install -r requirements.txt

---

## Step 5 - Run Streamlit App

Run:

streamlit run app.py

---

## Step 6 - Open Browser

Streamlit will automatically open browser.

If not, open:

http://localhost:8501

---

# Deploy on Streamlit Cloud

## Step 1
Upload project to GitHub

## Step 2
Open:
https://streamlit.io/cloud

## Step 3
Connect GitHub Repo

## Step 4
Deploy app.py

Your billing software will go live.
